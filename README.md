# Getting Started
## Install locally
Make a virtual environment:
```shell
python -m venv venv
venv\Scripts\activate.bat
```
Install dependencies
```shell
pip install -r requirements.txt
pip install -r requirements-dev.txt
pre-commit install
pip install -e .
```

Run tests to see everything working
```shell
pytest
```
